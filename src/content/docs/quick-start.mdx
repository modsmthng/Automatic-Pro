## Choose the right release track

- Start with [Automatic Pro v2](../v2/) if you want the safest baseline.
- Use [Automatic Pro vIT3](../vit3/) only if you deliberately want to test the newer saturation and extraction logic.
- Pick the file that matches your basket or dose. The file names already reflect the intended dose and any special basket type.

## Default starting point

Automatic Pro v2 defaults to:

- `91°C`
- `1:2 ratio`
- Classic espresso with medium to dark roasts

For many setups, the **18g v2 profile** is the easiest baseline because it is the reference profile behind most of the written guidance.
