## Quick notes

- `vIT3` is the current testing family and is expected to become **Automatic Pro v3** after the test phase.
- Updates can affect only some variants, so the file list above shows the latest build number for each currently published download.
- Use `v2` if you want the safest recommendation; use `vIT3` if you want to test the newer saturation and extraction logic.
- Discussion and feedback on [GaggiMate Discord](https://discord.com/invite/3JcR5csD4E).
